# 仓库查询功能使用说明

## 功能概述

仓库查询功能允许用户查看、筛选和复制仓库地址信息。

## 数据准备

### 1. 解析Excel文件

运行以下命令解析 `AD.xlsx` 文件并生成仓库数据：

```bash
# 首先安装依赖
pip install openpyxl

# 运行解析脚本
python parse_warehouse.py
```

脚本会：
- 读取 `AD.xlsx` 文件
- 解析第一行作为表头
- 从第二行开始读取数据
- 生成 `data/warehouse_data.js` 文件

### 2. 数据格式

生成的 `warehouse_data.js` 文件格式如下：

```javascript
window.warehouseData = {
  "warehouses": [
    {
      "代码": "RDU4",
      "地址": "1234 Warehouse St",
      "城市": "Raleigh",
      "州/省": "NC",
      "邮编": "27601",
      "国家": "USA",
      "类型": "FBA"
    },
    // ... 更多仓库数据
  ],
  "total": 4
};
```

## 功能特性

### 1. 搜索功能
- 在搜索框中输入关键词，可以搜索所有字段（代码、地址、城市、州/省、邮编、国家等）
- 支持实时搜索，输入即筛选

### 2. 筛选功能
- 可以按"类型"筛选仓库
- 筛选选项会根据数据自动生成

### 3. 选择功能
- 可以单独选择仓库（点击复选框）
- 可以全选当前筛选结果中的所有仓库
- 显示已选中的仓库数量

### 4. 复制功能
- **单个复制**：点击每行的复制按钮，复制该仓库的完整信息
- **批量复制**：选中多个仓库后，点击"复制选中"按钮，复制所有选中仓库的信息

### 5. 复制格式

复制的信息格式为：
```
代码: RDU4 | 地址: 1234 Warehouse St | 城市: Raleigh | 州/省: NC | 邮编: 27601 | 国家: USA | 类型: FBA
```

## 使用方法

1. 在"快速查询"Tab中点击"仓库查询"按钮
2. 在搜索框中输入关键词进行搜索
3. 使用下拉菜单按类型筛选
4. 勾选需要复制的仓库
5. 点击"复制选中"按钮复制信息

## 注意事项

- 如果 `AD.xlsx` 文件不存在或格式不正确，解析脚本会报错
- 确保Excel文件的第一行是表头
- 数据文件 `warehouse_data.js` 需要在页面加载前引入
- 如果数据未加载，页面会显示错误提示
